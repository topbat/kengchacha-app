## 一、整体调性（Creative Direction）

| 维度 | 设定 |
|------|------|
| 气质 | 官方、克制、高级感；科技感与温度并存（既严谨又有人情味） |
| 视觉 | 大留白、克制动效、动态排版（kinetic typography）为主，真机截图为证 |
| 主色 | 品牌青绿 `#0E9E8E` → 深青 `#0B6E6B` 渐变；薄荷绿 `#16D6B4` 点缀；深墨 `#0F172A` 文字；浅灰 `#F4F7FA` 背景 |
| 字体 | 思源黑体 / 苹方 / 微软雅黑（无衬线、字重对比强）；数字用等宽 |
| 转场 | 统一「淡入 + 轻微上浮（translateY 16px）」；模块间用品牌色块滑入分隔 |
| 配乐 | 轻电子 + 钢琴，BPM 90–100，干净不喧宾夺主；高潮在 Act3 结尾与收尾 |
| 旁白 | 沉稳男声或知性女声，普通话标准；语速适中、句末略收 |
| 安全 | 全程不点名具体企业；涉 AI 处标注"不构成法律/官方结论" |

> 落地物：本仓库 `promo/index.html` 是与本脚本同步的**可自动播放动态宣传片**（用真机截图 + 品牌动效），可全屏播放或录屏导出为视频；如需配音/配乐，按下表旁白逐句录制后叠加即可。

## 四、旁白全文（连读 · 配音用）

> 每一天，我们都在和形形色色的"坑"擦肩而过。看不清、问不到、来不及，往往就中了招。
> 现在，多了一个随身的"防坑入口"。坑查查——把反诈、防坑、消费维权的常识，做成可查、可测、可分享、可守护。
> 遇到可疑信息，第一时间，能看、能查、能问、能测、能上报，还能守护家人。一套产品，移动端与 PC 端，体验如一。
> 「避坑头条」，五维筛选、智能推荐，资讯每两小时滚动更新。
> 核心的「风险检测工具箱」——合同体检自动标红，链接验毒、截图识坑、收款核验，四合一，秒出风险分，记录全程私密。
> 拿不准，就问「AI 避坑助手」：判定风险、拆解套路、给出下一步与官方渠道。
> 「防坑自测」覆盖六大维度，测出你的防坑画像与高危场景。
> 踩过的坑也能帮到别人：语音说出经过，AI 自动成稿，一键上报到踩坑广场。
> 我们更想守护家里的长辈——为家人订阅风险领域，一键生成适老化口播预警。
> 每一次学习与上报，都会沉淀为你的防坑贡献值与徽章。
> 一键生成科普海报，全程语音可听可说，可解释的个性化推荐，PC 端同样从容。
> 坑查查构建在严谨的工程底座之上：Java 21、Spring Boot、React 双端响应式；离线可跑，云能力一键切换，稳且可成长。
> 看得清，查得到，测得准，护得住。坑查查。出品：topbat。
