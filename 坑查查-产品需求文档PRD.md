## 1. 产品概述

### 1.1 产品定位
一款面向中国普通大众、覆盖**从小到大、从生活到工作全场景**的「生活防坑查询 + 能力自测 + 实时风险检测 + 互助科普」平台。通过**真实经历沉淀 + 官方权威预警 + AI 智能加工**三类内容源，帮助用户识别并规避生活中的各类陷阱（侥幸心理、法律盲区、求职诈骗、消费踩坑、金融骗局、养老骗局等），并通过趣味自测、AI 对话、拍照/链接体检等方式，把"防坑常识"变成"随手可查、张口能问、动手能测"的国民级工具。

一句话定位：**"有坑没坑，先查查；遇事不决，问坑查查。"**

### 1.2 核心价值
- **真实性与时效性**：内容仅来源于"真实经历 UGC + 官方权威预警 + 判例/投诉数据"，并经 AI 与人工双重核验；前台内容池**每 2 小时滚动更新**，紧跟最新骗局变种。
- **极致低门槛**：不会打字也能用——支持**语音输入、语音搜索、AI 对话查询、拍照识别、一键体检**；提供**适老化模式**（大字、语音优先、家人代查）。
- **视觉化阅读**：核心要素由系统自动生成**警示海报、动态 GIF、信息长图**，重点数据与套路高亮，降低阅读门槛。
- **AI 驱动**：从内容生产、智能问答、自适应测试、风险检测到个性化画像，全链路由 AI 赋能，做到"千人千面、即问即答、即拍即识"。
- **互动与激励**：自测生成"防坑画像"，UGC 分享获得稀缺身份徽章，形成社区互助闭环。

### 1.3 目标用户与典型场景

|------|----------|--------------|

## 2. 设计原则

1. **降低门槛优先（Low-friction First）**：任何核心功能都至少提供"文字 + 语音/拍照"两种以上输入方式；老人、视障、不擅打字者均可独立使用。
2. **真实与权威（Truth & Authority）**：不编造、不夸大；每条内容标注来源类型（真实经历/官方预警/判例投诉），可溯源。
3. **AI 增益而非 AI 噪音（AI as Augmentation）**：AI 用于"提效与个性化"，但**关键事实经人工/规则复核**，避免误导。
4. **正向激励，不制造焦虑（Empower, not Fear-monger）**：内容以"教会用户怎么做"为落点，每个坑都给出可执行避坑动作。
5. **隐私与合规为底线（Privacy by Design）**：自测结果、上报私事默认私密、加密存储；佐证图片自动打码。

### 维度四：损失类型与危害等级（Impact）
- **损失类型**：金钱 / 健康 / 时间精力 / 个人隐私 / 法律责任 / 信用征信 / 人身安全。
- **危害等级**：🟢 轻度（小额可挽回） / 🟡 中度（较大损失/难追回） / 🔴 重度（重大财产、健康或人身/法律风险）。
- **损失金额区间**（用于筛选与统计）：<1k / 1k–1w / 1w–10w / >10w。

## 5. 详细功能需求

### 5.1 避坑头条（内容展示与检索）
**目标**：高频更新、视觉丰富、要素突出的真实避坑案例库。

- **2 小时滚动更新机制（由 6 小时调整）**
  - 系统每 **2 小时**对前台展示内容流进行一次轮换刷新，保持新鲜度与对新型骗局的跟进速度。
  - 前台显示「距下次更新：XX:XX」倒计时；支持下拉手动刷新拉取增量。
  - 更新需对 PC 与 H5 强一致、无感知、无卡顿（详见 §11、技术文档 §9）。
- **四要素提炼（强制）**：每条内容必须提炼 **【踩坑人群】【坑人套路】【损失后果】【避坑常识】**，由 AI 抽取 + 人工抽检。
- **极简文本**：正文 ≤ 200 字，直击痛点；附"一句话避坑口诀"。
- **视觉强化**：每条内容自动生成**动态 GIF / 警示海报 / 信息长图**，关键数据与套路高亮。
- **内容溯源标签**：标注来源类型（真实经历 / 官方预警 / 判例投诉）与发布时间、热度。
- **分类与检索**：
  - 按 §3 五维标签（人生阶段、领域、心理机制、危害等级、渠道）多维组合筛选。
  - **关键词全局搜索**，并支持**语音搜索**（说出"装修增项怎么避"即可检索）。
  - **AI 语义搜索**：自然语言→语义匹配最相关案例（如"我妈被拉进一个荐股群"）。
- **语音播报（TTS）**：任意内容一键朗读，适配老人/视障/通勤场景，支持倍速与方言（普通话优先，逐步扩展）。
- **个性化信息流**：登录用户基于防坑画像与地域，AI 推荐高相关、高危内容置顶。

### 5.2 AI 避坑助手（新增｜对话 + 语音）
**目标**：把"查资料"升级为"问一句就懂"，是全平台最低门槛的入口。

- **多模态输入**：文字 / 语音（ASR）/ 拍照截图（OCR + 图像理解）/ 粘贴链接。
- **对话式查询**：用户用大白话提问，助手基于平台内容库做 **RAG 检索增强问答**，给出：
  1. 这是不是坑、属于哪类套路（关联 §3 分类）；
  2. 对方下一步可能怎么做（套路推演）；
  3. 你现在该怎么做（可执行步骤 + 维权/报案渠道）；
  4. 关联真实案例卡片（可点开看头条原文）。
- **语音对话**：全程可"说话问、听回答"（ASR + LLM + TTS），适老/视障友好。
- **场景化追问**：助手会主动追问关键信息（"对方是否要你先付款？""有没有签合同？"）以精准判断。
- **安全护栏**：
  - 不提供任何"如何实施诈骗/规避监管"的内容（仅防御视角）。
  - 涉及报警、法律、人身安全时，给出 **96110 反诈专线、110、12315、12348 法律援助**等官方渠道，并提示"以官方机构核实为准"。
  - 对不确定问题明确说明不确定，引导人工/官方核实，杜绝编造。
- **一键转测/转上报**：对话中可直接"测测我会不会中招"或"把我的经历记下来帮别人"。

### 5.3 防坑能力自测与画像
**目标**：通过测试给出个性化防坑能力评估与建议。

- **出题机制**：
  - 规模：**快测 10 题 / 标准 30 题 / 深度 50 题**（V1 的 30/50 基础上增加 10 题快测，进一步降门槛）。
  - 模式：随机出题 / 针对性出题（按人群标签定向）/ **AI 自适应出题（IRT 难度自适应，答对升难、答错补弱项）**。
  - 维度覆盖：心理陷阱、法律常识、消费防骗、金融理财、职场求职、网络诈骗、医疗健康、家庭/养老等（对齐 §3）。
  - **题目可语音朗读、可语音作答**（说"A/第一个"即可）。
- **即时分析与画像**：
  - 秒级计算总分与各维度短板。
  - 生成趣味且精准的**防坑画像标签**（如"易信人小白""法网外游侠""沉没成本猎物""贪促销体质""权威易感人"等）。
  - **雷达图**展示各维度能力；**预测 3 个高危踩坑场景**；给出**具体可执行的防范动作**（"签合同前必查企业资质""拒绝先付款的兼职""转账前用本平台链接验毒"）。
- **隐私与分享**：默认私密、加密存储、仅本人可见；可一键生成"个性化分享卡片"，隐去敏感得分，仅展趣味画像与防坑宣言。
- **历史与成长**：可多次测试，查看能力提升曲线，针对短板推荐对应头条内容与微科普。

### 5.4 实时风险检测工具箱（新增）
**目标**：把"事后科普"升级为"事前拦截"，做用户的"随身安检仪"。

| 工具 | 输入 | AI 能力 | 输出 |
|------|------|---------|------|
| **合同体检** | 拍照/上传劳动/租房/借款/装修等合同 | OCR + 合同条款风险审查 | 标红风险条款（如"空白条款""违法竞业""定金不退""高额违约金"），逐条解释与建议 |
| **链接验毒** | 粘贴可疑链接/二维码 | 钓鱼/仿冒/恶意域名检测 + 内容研判 | 风险等级、仿冒对象、处置建议（勿填信息/勿转账） |
| **拍照识坑** | 拍/传截图（聊天、招聘、广告、收款页） | 图像理解 + 话术识别 | 命中套路类型、危险话术高亮、下一步建议 |
| **收款/账户核验** | 输入对方提供的"客服话术/收款方式" | 规则库 + LLM 研判 | 是否符合典型诈骗特征（如"对公转私""走私人账户""屏幕共享") |
| **企业/招聘核验**（导流） | 公司名称 | 关联公开工商/失信/投诉信息提示 | 风险提示并引导至官方平台（国家企业信用信息公示系统等）核验 |

> 所有工具均明确标注"AI 辅助判断，不构成法律/官方结论"，并提供官方核验入口。检测记录默认私密、加密。

### 5.5 一键分享引擎
**目标**：降低传播门槛，支持内容与自测结果的社交裂变。

- **素材自动生成**：单条避坑经历、自测画像、风险检测结果均可自动合成精美长图海报（适配移动端比例，含平台水印与防伪二维码）。
- **多端分享通道**：
  - 小红书：一键保存图片 + 复制带话题文案（#生活避坑指南 #防坑自测 #坑查查）。
  - 微信：一键保存高清海报 / 生成 H5 卡片直发好友与朋友圈。
  - 其他：微博、QQ、抖音（保存导出）。
- **分享落地页**：强引导"测测你会中什么坑""问问 AI 避坑助手""查看更多真实案例"，承接新用户进入主流程。

### 5.6 踩坑经历上报与身份激励（UGC + 成长体系）
**目标**：鼓励贡献真实经历，以稀缺身份标识构建正向点赞氛围。

- **极简上报**：表单含时间、地点、人群标签、领域、事件经过、避坑建议；
  - **支持语音转文字一键填写**、拍照上传佐证（系统**自动人脸/证件/隐私打码**）。
  - **AI 辅助成稿**：用户口述要点，AI 自动整理为"四要素"结构化草稿，用户确认即可提交（大幅降门槛）。
- **审核机制**：**AI 初审（涉政涉黄涉敏、虚假、广告、辱骂、个人信息）+ 人工复审**双重过滤；驳回给出原因与修改建议。
- **贡献激励**：
  - 每条通过审核的有效记录奖励**防坑贡献值**。
  - **稀缺身份徽章体系**（高级感 + 稀缺性）：如【避坑先驱】【人间清醒·LV3】【反诈布道者】【打假斗士】等，动态徽章随等级进阶。
  - 徽章展示于个人主页、评论、上报记录显眼处。
- **社交点赞**：浏览经历可"点赞（学到了）""点亮（同情/抱抱）""收藏"；高级身份用户的内容享**专属流光边框**，激发点赞与认同，形成正向激励。
- **防刷与防滥用**：限制频率、查重去重、反作弊（详见技术文档 §10）。

### 5.7 家人守护（新增）
**目标**：让年轻人替家中老人/孩子"远程防坑"，覆盖最易受害却最不会用 App 的人群。

- **亲情账户绑定**：子女与父母账号建立守护关系（双向确认）。
- **代查代问**：子女可代老人发起查询、合同体检、链接验毒，并把结果以**大字/语音卡片**推送给老人。
- **风险提醒**：可为家人订阅其高发坑点（如"养生保健品""以房养老"），平台一旦有同类新预警，推送给守护人。
- **一键求助**：老人端提供醒目"问问孩子/打电话"按钮与一键拨打 96110。
- **隐私边界**：仅共享风险类信息，不窥探隐私内容；所有共享需被守护人授权。

### 5.8 个人中心
我的防坑画像、身份徽章墙、我的上报与审核状态、我的点赞收藏、我的检测记录、家人守护关系、消息通知、隐私与安全设置（含数据导出/删除）。

### 5.9 无障碍与适老化 / 语音无障碍层（贯穿全局）
- **语音优先模式**：搜索、提问、答题、上报全部可"说"；回答可"听"。
- **适老化模式**：超大字号、高对比、简化导航、语音引导、家人代查入口前置。
- **多模态入口**：文字 / 语音 / 拍照 / 链接四类入口在关键页面统一暴露。
- **无障碍合规**：兼容读屏，符合信息无障碍要求。

## 6. 内容来源与运营策略（含数据来源判断）

> 用户希望"尽可能真实、全面、贴近中国普通人全生命周期的坑"，以下为平台**合规可用**的内容来源判断与采集策略。**原则：优先公开的官方/权威预警与脱敏聚合数据，UGC 经审核，严禁抓取并原样转载受版权或隐私保护的内容。**

### 6.1 权威与官方源（高可信，优先）
- **国家反诈中心**（预警案例、骗局类型科普）、**公安部刑侦/各地网警与反诈公众号**通报。
- **中国互联网联合辟谣平台**、**国家市场监督管理总局**消费警示、**中国消费者协会**消费提示与年度报告。
- **12315 全国消协投诉平台**、**12321 网络不良与垃圾信息举报中心** 公布的典型案例。
- **中国裁判文书网**（诈骗、合同纠纷、民间借贷等公开判例，做脱敏与法律科普加工）。
- **银保监会 / 证监会 / 央行**风险提示（非法集资、虚拟币、套路贷等）。
- **人民网 / 新华网 / 央视《焦点访谈》《3·15 晚会》** 等主流媒体的曝光报道（引用 + 注明出处，不整篇搬运）。

### 6.2 社会舆情与投诉源（聚合趋势，需清洗脱敏）
- **黑猫投诉、聚投诉** 等公开投诉平台的趋势与高频问题（做聚类统计与脱敏，不直接搬运个人投诉原文）。
- **知乎、豆瓣、小红书、贴吧** 等社区中"踩坑经历"类公开讨论（仅作为线索发现，AI 提炼共性套路后**重写成结构化科普**，不复制原文、不带个人隐私）。
- 各地市场监管局"红黑榜"、行业协会通报。

### 6.3 平台自有 UGC（独家、真实）
- 用户上报的真实经历，经审核脱敏后纳入内容池，是平台差异化的核心资产。

### 6.4 采集与加工策略
1. **多源采集**：官方源以 API/RSS/合规抓取为主；社区源仅做"线索发现"，不存原文。
2. **AI 清洗与改写**：去广告、去隐私、提炼"四要素"、重写为原创科普文本（规避版权与隐私风险）。
3. **真实性与去重**：交叉核验、相似聚类去重、可信度评分；低可信内容不进前台。
4. **合规过滤**：内容安全审核（涉政涉敏涉黄、谣言、点名攻击）→ 人工抽检。
5. **更新节奏**：每 **2 小时**生成/轮换一批前台内容（详见技术文档 §6、§9）。

> **合规边界声明**：平台只做"防御性科普与风险提示"，不点名定性具体企业为"诈骗"（除非有官方定性/生效判决），对未决事件使用"涉嫌/疑似/请核实"等审慎措辞，规避名誉侵权。

## 11. 非功能性需求

- **定时更新稳定性**：2 小时轮换采用高可用分布式调度 + 消息队列，PC/H5 数据强一致、更新无感知无卡顿；失败可重试与回滚。
- **视觉素材自动化**：后端文生图引擎将结构化文本自动套模板生成海报/GIF/信息长图，排版在 PC/H5 多分辨率不错乱。
- **性能**：头条首屏 < 1.5s；搜索/筛选 < 500ms；AI 助手首字响应 < 2s（流式）；自测画像秒级。
- **多端适配**：H5 完美适配主流机型浏览器与微信内嵌环境；PC 兼容主流分辨率。
- **可用性**：核心服务可用性 ≥ 99.9%；AI 服务降级有兜底（如助手不可用时回退关键词检索）。
- **可扩展**：内容量级支持千万级，向量检索毫秒级。

## 12. 隐私、安全与合规

- **隐私 by Design**：自测结果、上报私事、检测记录默认私密、**加密存储**，未授权绝不外公开。
- **个人信息保护**：遵循《个人信息保护法》《数据安全法》，最小必要收集、明示同意、可导出/可删除；佐证图片自动打码（人脸/证件/车牌/住址）。
- **未成年人保护**：未成年相关内容审慎处理，引导监护人；不收集非必要未成年人信息。
- **内容合规**：全程内容安全审核（涉政涉敏涉黄、谣言、点名攻击）；对未决事件审慎措辞，避免名誉侵权。
- **AI 合规**：使用境内合规大模型与算法备案；AI 生成内容显著标识"AI 辅助，不构成法律/官方结论"。
- **数据来源合规**：遵守 robots 协议与各源使用条款，社区源仅取线索、改写原创，不搬运侵权内容。

## 14. 风险与挑战

| 风险 | 影响 | 应对 |
|------|------|------|
| 内容版权与名誉侵权 | 法律风险 | 社区源只取线索、AI 改写原创；未决事件审慎措辞；建立投诉/删除机制 |
| AI 误判（合同/链接/对话） | 误导用户 | 明确"AI 辅助、非结论"，提供官方核验入口，关键场景人工兜底 |

| 隐私泄露 | 合规与信任 | 字段级加密、自动打码、最小收集、权限审计 |
| 2 小时更新带来的成本与一致性 | 稳定性/成本 | 高可用调度 + 缓存 + 灰度上架 + 监控告警 |
| 老人/视障真实可用性 | 目标人群流失 | 语音优先、家人守护代查、可用性实测迭代 |
