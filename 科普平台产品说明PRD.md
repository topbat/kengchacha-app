产品需求文档（PRD）：「避坑雷达」生活防坑与常识科普平台
1. 产品概述
1.1 产品定位
一款基于PC/H5双端的生活防坑查询与自测工具，旨在通过真实经历的沉淀与分享，帮助用户识别生活中的各类陷阱（如侥幸心理、法律盲区、求职诈骗、消费踩坑等），并通过趣味自测与个性化建议提升用户的防范意识。

1.2 核心价值
真实性与时效性：仅展示真实经历，每6小时滚动更新内容池。
视觉化阅读：核心要素通过海报、动态图、信息图呈现，降低阅读门槛。
互动与激励：通过自测生成防坑画像，通过UGC分享踩坑经历获取高价值身份标识，形成社区互助闭环。
2. 产品架构与模块划分
产品整体划分为四大核心模块：

避坑头条（内容展示与检索模块）
防坑能力自测与画像模块（分析与诊断模块）
一键分享引擎（社交裂变模块）
踩坑经历上报与身份激励模块（UGC与成长体系模块）
3. 详细功能描述
3.1 避坑头条（内容展示与检索模块）
功能目标：提供高频更新、视觉丰富、要素突出的真实避坑案例库。

6小时滚动更新机制：
系统每6小时对前台展示的内容流进行一次轮换与刷新，保持内容新鲜度。
用户端显示“距下次更新：XX小时XX分钟”倒计时。
内容呈现规范：
要素提炼：每条经历强制提炼四大要素：【踩坑人群】、【坑人套路】、【损失后果】、【避坑常识】。
极简文本：正文描述控制在200字以内，直击痛点。
视觉强化：每条内容必须配有系统自动生成的动态GIF、警示海报或信息长图，将重点数据与套路高亮显示。
分类与检索：
支持按地区、人群（如学生、宝妈、老人、求职者）、标签（如法律边缘、兼职诈骗、消费陷阱、填报志愿）进行精准筛选。
支持关键词全局搜索。
3.2 防坑能力自测与画像模块
功能目标：通过随机或有针对性的题目测试，为用户提供个性化的防坑能力评估与生活建议。

题库与出题机制：
支持“随机出题”与“针对性出题”（根据用户选择的人群标签定向出题）。
每次测试生成 30 题或 50 题的问卷，涵盖心理陷阱（如侥幸心理）、法律常识、消费防骗、职场求职等维度。
即时分析与画像生成：
用户提交后，系统秒级计算得分与各维度短板。
生成防坑画像：赋予趣味且精准的标签（如“易信人小白”、“法网外游侠”、“沉没成本猎物”等）。
预测高危踩坑场景：列出用户未来极有可能中的3个坑。
生活与工作建议：给出具体可执行的防范动作（如“签订合同前必查企业资质”、“拒绝先付款的兼职”等）。
隐私与分享设置：
默认私密，仅用户本人可见。
支持一键转为“个性化分享卡片”，隐去敏感得分，仅展示趣味画像与防坑宣言，供社交平台分享。
3.3 一键分享引擎
功能目标：降低传播门槛，支持内容与自测结果的社交裂变。

分享素材自动生成：
无论是单条避坑经历，还是自测画像，系统均自动合成精美长图海报（适配移动端屏幕比例）。
多端分享通道：
小红书分享：一键调起或保存图片并复制带话题的文案（如 #生活避坑指南 #防坑自测）。
朋友圈分享：一键保存高清水印海报，或生成H5链接卡片直接发送至微信好友/朋友圈。
3.4 踩坑经历上报与身份激励模块
功能目标：鼓励用户贡献真实经历，通过高级身份标识构建点赞社交氛围。

踩坑经历上报（UGC）：
用户可填写自己曾经踩过的坑，表单需包含：时间、地点、人群标签、事件经过、避坑建议。
支持上传相关佐证图片（打码处理）。
审核与身份标识奖励机制：
系统与人工双重审核，过滤虚假/违规信息。
有效记录奖励：每成功通过审核发布一条有效记录，用户获得“防坑贡献值”。
高级身份标识体系：
达到不同贡献值解锁专属动态身份徽章，标识设计需具备“高级感”与“稀缺性”，如：【避坑先驱】、【人间清醒·LV3】、【反诈布道者】等。
标识展示在用户的个人主页及所有评论、上报记录的显眼处。
社交点赞机制：
其他用户在浏览经历时可进行“点赞（学到了）”、“点亮（同情）”等互动。
拥有高级身份标识的用户，其上报的经历享有专属流光边框，更容易唤醒普通用户的点赞与崇拜心理，形成正向激励。
4. 访问路径与页面流转
4.1 PC端路径规划
首页（避坑头条信息流）
路径：/home
包含：6小时更新倒计时、分类导航、图文/动图信息流。
防坑自测中心
路径：/test
流程：选择测试规模（30/50题） -> 选择模式（随机/针对） -> 答题页 -> 提交计算 -> 画像结果与建议页 -> 隐私设置/分享面板。
踩坑经历广场（UGC库）
路径：/stories
包含：用户上报的真实经历列表，按热度或时间排序，展示上报者的高级身份标识。
个人中心
路径：/profile
包含：我的防坑画像、我的身份标识墙、我上报的记录及审核状态、我的点赞收藏。
4.2 H5端路径规划（适配微信/浏览器环境）
启动页：动态加载动画（雷达扫描动效），进入首页。
底部Tab导航：
【避坑头条】：对应PC首页信息流，支持下拉刷新获取最新6小时内容。
【能力自测】：快捷入口，卡片式答题体验，左右滑动切换题目。
【上报踩坑】：极简表单填写入口，支持语音转文字输入。
【我的】：展示专属高级徽章墙、我的测试画像历史、分享记录。
分享落地页：
路径：/share/{content_id} 或 /share/{test_result_id}
页面设计侧重视觉冲击力，底部强引导“测测你会中什么坑”或“查看更多避坑经历”，引导新用户进入主流程。
5. 非功能性需求
定时任务稳定性：6小时更新机制需采用高可用队列，确保PC与H5端数据强一致性，更新无感知、无卡顿。
视觉素材自动化：后端需接入图像生成引擎，能够将结构化文本（要素、建议）自动套用模板生成精美的H5海报与GIF动图。
数据隐私保护：自测结果及用户上报的私人经历在未授权分享前，必须进行加密存储，绝不对外公开。
多端适配：H5需完美适配各主流机型浏览器及微信内嵌环境，PC端需兼容主流分辨率，确保海报排版不错乱。
